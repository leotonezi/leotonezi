## 🔨 building

**[pet-tamagotchi](https://github.com/leotonezi/pet-tamagotchi)** — on-chain tamagotchi on solana. virtual pet with real-time stat decay and death mechanics, built with anchor and rust.

**[sandslash](https://github.com/leotonezi/sandslash)** — async seo auditing cli in rust. crawls sites, runs checks, outputs scored json reports.

**[sonic-library](https://github.com/leotonezi/sonic-library)** — digital library with ai-powered book recommendations via langchain and openai.

**[amigo-agiota](https://github.com/leotonezi/amigo-agiota)** — subscription payment tracker. participants upload proof of payment monthly, owner approves. no more whatsapp chaos.
